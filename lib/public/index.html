Book Shoppe

This API uses lists of books and authors as its resources. The routes are as follows:
Route Method Description Example 
/books
 GET Gets a list of books' names, ids and release-dates List Books 
/books/:id
 GET Returns the detail about the book matching the id in the parameter Get Book 1 
/authors
 GET Gets a list of authors' names and ids List Authors 
/authors/:id
 GET Returns the detail about the author matching the id in the parameter Get Author 1
